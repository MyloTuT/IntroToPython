Your code will only implement the by_openclose function. Please see the homework discussion for details. Special note: a common misconception is that the sort function should accept the list of elements as an argument and return a sorted list of elements. Sort funtions don't work with a list. Instead, they take one value (an item to be sorted) and return one value (the value by which it should be sorted), not a list of values. The returned value is the "value by which this element should be sorted" in order to inform the sorting that sort() will do.
